# FII Logger

![GitHub](https://img.shields.io/github/license/federation-interservices-d-informatique/fiibot-logger?style=for-the-badge)

Bot de logs de la fii

## Commandes

### Logchan

#### Utilisation

```
/logchan
```

Retourne le canal de logs du bot

```
/logchan #channel
```

Set the log channel
